# Profile-Website

This is a simple, personal website using HTML and CSS only. A project to build a main landing page, separate About page with proper styles, assets and real world Github workflow. A week 2 and 3 practice assignment on Git with Bash command, GitHub, HTML and CSS, whose objective is to help us practice and demonstrate our understanding of the technologies learnt so far.
The website contains personal details such as full name, email, contact and others (age, location, occupation and hobbies) displayed in a list format.
The project structure is as follows:

``` See Below the Folder Structure:

Profile-Website/
│
├── src/
|   |── Pages
|   |   ├── About.html
|   |   ├── Contact.html
|   |   ├── Home.html
|   |   ├── Profile.html
|   |   ├── Skills.html
│   ├── CSS/
│   │   └── Style.CSS
│   └── Images/
│       └── profile.jpg
├── index.html
└── README.md
```

## Features

- Personal details displayed in a list format.

- A simple, personal website using HTML and CSS only.

##

Getting Started
To view the website, simply open the `index.html` file in your web browser. Additional setup is required.
